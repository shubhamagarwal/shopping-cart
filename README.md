## Prerequisites

```
node (version > 10)
npm

```

## Tech Stack

```
React JS
JavaScript
Redux
Enzyme
chai
etc
```

# Run development project locally

```
Clone https://github.com/shubhamagarwal/shopping-cart.git
npm install
npm run start
open http://localhost:3000 to view project on the browser
```


# Run Test Cases

```
npm run test
```
